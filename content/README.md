# Content

Distribution copy for every learnrudi article, organized by piece.

Each folder maps to one article. Inside: one file per channel, plus a `meta.md`
that tracks publish status and links.

The article HTML lives in `public/insights/`. This folder holds what you do with it.

## Structure

```
content/
  <article-slug>/
    meta.md        ← status, URLs, dates
    linkedin.md    ← post copy
    substack.md    ← adapted prose
    tiktok.md      ← scripts (one per clip)
    youtube.md     ← outline + title options
```
