## Post

There's a category of work that nobody puts on their calendar but everyone does every week.

Logging the email in the CRM. Turning meeting notes into tasks. Filing the invoice after the payment clears. Moving information from the tool where it landed to the tool where it lives.

It doesn't feel like real work. But it costs real time. And it compounds.

What changed recently is that most of these tools now speak the same language — a protocol called MCP that lets an AI agent move between them without custom code. That means the handoff that used to require a developer, a Zapier subscription, or just you doing it manually can now run automatically, triggered by the event that caused it.

I put together seven of the most common ones:

**01 — Meeting → Action Items** (Otter.ai + Notion / Todoist)
Recording ends. AI reads the transcript, pulls every action item, assigns owners, sets dates. Tasks appear in your project tool before you've closed the Zoom window.

**02 — Email → CRM Update** (Gmail + HubSpot / Salesforce)
Prospect replies. AI reads the thread, updates the deal stage, logs the interaction, queues a follow-up draft. Rep opens the CRM and it's already done.

**03 — Payment → Bookkeeping** (Stripe + QuickBooks / Xero)
Payment succeeds. AI categorizes the transaction based on the product and customer record, creates the invoice entry. Anything it can't match cleanly gets flagged for review.

**04 — Alert → Incident Briefing** (Datadog + PagerDuty + Slack)
Something breaks. AI pulls the anomaly data, recent deploy history, and affected services into a Slack post. Engineer wakes up to context, not a dashboard hunt.

**05 — Signed Contract → Onboarding Project** (PandaDoc + Asana / Linear)
Signature lands. AI reads the contract, extracts scope and timeline, creates the onboarding project with tasks and milestones pre-filled. No inbox delay.

**06 — Prospect Research → Personalized Outreach** (Apollo / LinkedIn + Gmail)
Target name goes in. AI pulls role, recent activity, company signals. Draft comes out opening on something specific — a post, an announcement, a hiring pattern.

**07 — New File → Knowledge Base Entry** (Google Drive + Notion)
File lands in a shared folder. AI summarizes it and creates a linked Notion entry with tags. Findable the same day it's uploaded.

---

None of these require a developer. The setup cost for most is an afternoon. The MCP connectors handle the authentication. You write a short prompt describing what the AI should do. You test it once on a real example.

The question is just which manual step you're going to stop doing first.

Start with whatever you did this week.
