# Contributing to RUDI

Thank you for your interest in contributing to RUDI - Responsible Use of Digital Intelligence.

## About RUDI

RUDI is an education platform focused on AI literacy, responsible AI use, and professional certification. We help organizations build AI capacity through training, policy frameworks, and hands-on tools.

## How to Contribute

### Content Contributions

- **Curriculum suggestions**: Ideas for courses, modules, or learning paths
- **Resource recommendations**: Links to valuable AI literacy resources
- **Case studies**: Real-world examples of responsible AI adoption

### Technical Contributions

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test locally
5. Submit a pull request

### Website Improvements

- Bug fixes
- Accessibility improvements
- Performance optimizations
- Mobile responsiveness

## Code Style

- Use semantic HTML
- Follow existing CSS conventions
- Keep JavaScript simple and accessible
- Test across browsers

## Reporting Issues

When reporting issues, include:
- Description of the problem
- Steps to reproduce
- Browser and device information
- Screenshots if applicable

## Questions

For questions about contributing, open a GitHub issue or reach out through the website contact form.
