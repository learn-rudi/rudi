RUDI Brand Assets — Screenshot & Upload

Pick one icon and one logo. Screenshot each at exact pixel size. Upload to Stripe Dashboard → Settings → Branding.

How to capture (Mac):
  1. Press ⌘ + ⇧ + 4 (Command-Shift-4) — cursor becomes crosshairs
  2. Hold spacebar, hover over the asset block — it highlights, releases the spacebar OR draw a tight box around just the asset (the dashed border is OUTSIDE the asset — don't include it)
  3. Click to capture — PNG saves to your Desktop
  4. Upload to Stripe at dashboard.stripe.com/settings/branding
Quick screenshot tips:
  1. Use Chrome at 100% zoom (View → Actual Size, or ⌘ + 0) so pixel dimensions are exact
  2. If you want a tighter PNG, open the file at the exact pixel size — every asset block is sized to Stripe's recommended dimensions
