Documentation

Welcome to PromptStack! This guide will help you get started with the AI workstation for builders.

Installation

System Requirements

  • macOS: Version 12 (Monterey) or later
  • Memory: 4GB RAM minimum (8GB recommended)
  • Storage: 500MB available space
  • Internet: Required for AI provider connections

Download & Install

  1. Download the latest version from GitHub Releases
  2. Open the downloaded .dmg file
  3. Drag PromptStack to your Applications folder
  4. Launch PromptStack from Applications or Spotlight

First Launch Security

If macOS blocks the app, go to System Settings → Privacy & Security and click "Open Anyway" to allow PromptStack to run.

First Run

When you first open PromptStack, you'll see the main interface with three panels:

  • Left Panel: File Explorer - Browse and manage your project files
  • Middle Panel: Content Editor - View and edit files with syntax highlighting
  • Right Panel: Terminal & AI Chat - Execute commands and chat with AI agents

Setting Up AI Providers

To use AI features, you'll need to configure at least one provider:

  1. Open Settings from the top menu
  2. Navigate to the "AI Providers" section
  3. Enter your API key for Claude, Codex, Gemini, or another supported provider
  4. Save your settings

Supported providers:

  • Claude - Anthropic's AI assistant
  • Codex - OpenAI's code model
  • Gemini - Google's AI model
  • Ollama - Local AI models

Chat Interface

The chat interface lets you interact with AI agents while maintaining full context of your project.

Starting a Conversation

  1. Open the Terminal panel (right side)
  2. Click the "Chat" tab to switch to headless chat mode
  3. Select your AI provider from the dropdown
  4. Type your message and press Enter

Chat Features

  • File Context: Attach files to provide context to the AI
  • Tool Use: AI can execute tools and commands on your behalf
  • Streaming Responses: See AI responses in real-time
  • Thinking Blocks: View Claude's internal reasoning process
  • Session History: Access previous conversations

File Explorer

The File Explorer (left panel) provides a tree view of your project files.

Features

  • Quick Access: Pin frequently used folders for easy access
  • Fuzzy Search: Press Cmd+K to quickly find files
  • Inline Editing: Rename files directly in the tree
  • Drag & Drop: Reorganize files and folders
  • Context Menu: Right-click for file operations

Keyboard Shortcuts

  • Cmd+K - Open file search
  • Enter - Open selected file
  • Delete - Delete selected file
  • Arrow Keys - Navigate tree

Terminal

PromptStack includes a full PTY terminal emulator for running shell commands.

Features

  • Multiple Tabs: Run multiple terminal sessions simultaneously
  • Dev Server Detection: Automatically detect running development servers
  • File Attachments: Attach files to commands for AI context
  • Command History: Access previous commands with up/down arrows

Switching Between Chat and Terminal

Click the tabs at the top of the right panel to switch between:

  • Shell: Standard terminal with PTY support
  • Chat: AI chat interface with streaming

Git Integration

Troubleshooting

App Won't Open

Issue: macOS blocks the app from opening.

Solution:

  1. Go to System Settings → Privacy & Security
  2. Scroll down to find "PromptStack was blocked"
  3. Click "Open Anyway"
  4. Confirm by clicking "Open" in the dialog

AI Provider Not Working

Issue: API key not accepted or requests failing.

Solution:

  • Verify your API key is correct
  • Check your API key has sufficient credits/quota
  • Ensure you have internet connectivity
  • Try switching to a different provider

Files Not Showing

Issue: File Explorer is empty or files are missing.

Solution:

  • Check you've selected a project folder
  • Verify folder permissions
  • Try refreshing the file tree (Cmd+R)

Performance Issues

Issue: App is slow or unresponsive.

Solution:

  • Close unused terminal tabs
  • Clear old chat sessions
  • Restart the application
  • Check available memory on your system

FAQ

Is PromptStack free?

Yes! PromptStack is free during the beta period. Future pricing for cloud features will be announced.

What AI providers are supported?

PromptStack supports Claude (Anthropic), Codex (OpenAI), Gemini (Google), and Ollama for local models. More providers are being added.

Can I use my own API keys?

Yes! You bring your own API keys (BYOK) for all AI providers. This keeps your usage and costs under your control.

Does it work offline?

The file editor, terminal, and local features work offline. AI features require internet connectivity unless you're using Ollama with local models.

What file formats are supported?

PromptStack supports 33+ file formats including:

  • Code: JS, TS, Python, Shell, CSS, etc.
  • Documents: PDF, DOCX, Markdown, HTML
  • Data: JSON, JSONL, YAML, CSV, Excel
  • Media: Images (PNG, JPG, SVG), Video, Audio
  • System: Logs, SQLite databases, Archives

Can I customize the interface?

Yes! You can:

  • Resize panels by dragging
  • Collapse/expand sections
  • Customize keyboard shortcuts (coming soon)
  • Choose your preferred theme (coming soon)

Is my data private?

Yes. All your files and sessions are stored locally on your machine. When using AI features, only the messages you send are transmitted to your chosen AI provider according to their privacy policies.
